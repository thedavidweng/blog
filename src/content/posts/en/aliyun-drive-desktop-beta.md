**Aliyun Drive** is Alibaba's second attempt to join the cloud storage war after Kupan.

This time it carries the Alibaba prefix, symbolizing the determination to pour resources and its high internal status.

## Preface Summary

Currently, new users registered during the public beta period will have 500G of space, while internal beta users have 1T. If they also participated in the **Teambition Netdisk** test during the internal beta period, they should have received the 3T space after netdisk merger.

Welfare codes can be redeemed in the client sidebar **Welfare Club** entrance to obtain capacity expansion valid for one year.

## Multi-platform Support

The official website shows other versions are still under development.

## Beta Desktop Client

The beta version I got this time provided Windows/macOS dual versions, using Alibaba's signature, proving it is from Aliyun Drive official, not a third-party client.

![macOS installer package](/posts/aliyun-drive-desktop-beta/alidrive-desktop-macos-install.avif)


![Login screen](/posts/aliyun-drive-desktop-beta/alidrive-desktop-macos-login.avif)


![Desktop app main window](/posts/aliyun-drive-desktop-beta/alidrive-desktop-macos-ui.avif)


The functions provided by the beta client are the same as the web version. It is speculated to be a client made using **Electron**.

![Settings page](/posts/aliyun-drive-desktop-beta/alidrive-desktop-macos-settings.avif)


Currently, it supports adaptive dark mode. It does not support folder synchronization and album synchronization functions.

## Conclusion

The current desktop client is just a packaged web version. It does not add various functions common to other netdisk clients. It doesn't even include the album viewing/uploading function of the mobile client. It is only for early experience.
